# Mutability Syntax

```
// the generic signature is something like this
pub sig map! : (A -> B) -> mut F[A] -> F[B]

// "mut T" isn't really a type in this system,
// just a vague kind of effect annotation; this
// avoids adding new subtyping semantics

// fuck it; do I add a basic effects system here?

unsafe fn map!(f: A -> B, mut vec: Vec[A]) -> Vec[B] {
    for i in vec {
        set!(vec[i], f(vec[i]))
    }
}
```
